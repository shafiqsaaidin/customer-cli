# customer-cli
Command line interface fir managing customers. Uses Mongoose, COmmander.js and Inquirer.js

### Installation
`$ npm install`

### Create symlink
`$ sudo npm link`

### Remove symlink
`$ sudo npm unlink`

### Command List
- `client-cli list`
- `client-cli add`
- `client-cli update [_ID]`
- `client-cli remove [_ID]`
- `client-cli find [_ID or fname or lname]`
